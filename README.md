# login-with-jwt

Project ini adalah project backend

## How to Pakai??

Install git pada terminal cmd anda

Lalu ketikan :
``` $ git clone https://github.com/bolaxd/login-with-jwt.git```

Lalu Edit keperluan anda di file ```.env```

## How to Jalankan??
``` $ npm start```
Or
``` $ node index.js```
